What you'll learn

Our course is designed to give you the skills you need to excel in data analytics.


  • Soft Skills For Data Analysts

    The Soft Skills module is a specialized exploration into the non-technical competencies crucial for success in the field. Here, students will immerse themselves in honing vital interpersonal, communication, and problem-solving skills essential for effective data analysis, enabling data analysts to not only crunch numbers but also effectively convey insights, foster collaboration, and drive impactful decision-making across diverse professional landscapes.

  • Spreadsheets For Data Analytics & Statistics

    The Preparing Data module delves into data and spreadsheet expertise, using Google Sheets or Excel as our platform. Here, you'll learn how to efficiently retrieve, manipulate, and visualize data. The module is designed to equip you with the skills to draw actionable insights and make informed decisions, offering a comprehensive guide to data preparation and analysis.

  • Power BI For Dashboards & Insights

    The Data Visualisation and Storytelling module is all about using Power BI to show off your data. You'll learn how to get your data into the system, clean it up, and make it look good. By the end, you'll be able to take any set of numbers and turn them into a clear and engaging story.

  • SQL For Data Management

    In the SQL (Structured Query Language) module, you'll learn how to work with databases, which are the backbone of most big data projects. From the basics of how databases work to designing and running your own, you'll get a clear overview of what SQL is all about and why it's so important.

  • Python For Advanced Data Cleaning & Data Analytics

    With the Python module, you'll dive into one of the go-to languages for data analytics. You'll learn about the Notebook environment, work with different data types, and get hands-on with writing functions and data analysis. It's everything you need to start working confidently with Python.
